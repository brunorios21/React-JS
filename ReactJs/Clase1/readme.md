# Clase 1: Introducción a React y Configuración del Entorno

En esta primera sesión del curso de Talento Tech, sentamos las bases teóricas y técnicas necesarias para comenzar el desarrollo de aplicaciones modernas con React JS.

## Nivelación de Conceptos: HTML y CSS

Antes de profundizar en la lógica de componentes, realizamos un repaso estratégico de **HTML5** y **CSS3**. Estos lenguajes son los bloques fundamentales de construcción en la web:

* **Estructura Semántica:** Revisamos la importancia de etiquetas claras para el SEO y la accesibilidad.
* **Estilizado Moderno:** Conversamos sobre cómo CSS se integrará luego con React mediante módulos o frameworks.
* **Visión a Futuro:** Entendimos que estas bases son críticas para el consumo de **APIs** que desarrollaremos o integraremos en etapas avanzadas, donde la interfaz (Frontend) deberá mostrar datos dinámicos provenientes de un servidor.

---

## Introducción a React JS

Hablamos sobre la filosofía de React, enfocándonos en:
* **Componentización:** La capacidad de dividir la interfaz en piezas pequeñas y reutilizables.
* **Virtual DOM:** Cómo React optimiza el rendimiento actualizando solo lo que realmente cambia en pantalla.
