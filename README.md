## Descripción de ReactJS
ReactJS es una de las bibliotecas más relevantes en el desarrollo frontend moderno.  
Su importancia radica en la capacidad de crear interfaces de usuario dinámicas, escalables y altamente interactivas.  
Gracias a su ecosistema maduro (Hooks, Context API, React Router), permite resolver problemas comunes como la gestión de estado, autenticación, navegación y consumo de APIs.  
Aprender React no solo significa dominar una herramienta técnica, sino también adquirir competencias transversales en buenas prácticas de programación, arquitectura de software y diseño centrado en el usuario.  
En el ámbito laboral, React se ha convertido en un estándar de facto para startups, pymes y grandes empresas que requieren soluciones digitales ágiles.

## Objetivos
**General**  
Diseñar y desarrollar aplicaciones web interactivas con ReactJS, integrando componentes, estado, enrutamiento y autenticación.  

**Específicos**  
- Comprender fundamentos de ReactJS y su ecosistema  
- Manejar estado local y global (useState, Context API)  
- Crear rutas dinámicas y protegidas  
- Integrar formularios con validación y autenticación  
- Construir un proyecto final de e-commerce con CRUD  
- Aplicar buenas prácticas de estilización y despliegue  

---

## Resultados de Aprendizaje
Al finalizar el curso, el estudiante podrá:  
- Crear una aplicación con Vite  
- Implementar gestión de estado con useState y Context API  
- Desarrollar rutas dinámicas y protegidas  
- Construir formularios con validación y autenticación  
- Implementar CRUD de productos  
- Estilizar con Bootstrap o styled-components  
- Desplegar la aplicación en hosting (ej. Vercel, Netlify)

## Temario Programado

| Clase | Contenido |
|-------|-----------|
| 1 | Introducción a React: instalación con Vite y Node.js, estructura básica, componentes funcionales |
| 2 | JSX y componentes: sintaxis, props, patrón contenedor/presentacional |
| 3 | Estructura de componentes y navegación básica: Header, Footer, estilos con CSS Modules |
| 4 | useState: gestión de estado local, eventos, carrito de compras |
| 5 | useEffect y peticiones a APIs: carga de productos, manejo de errores |
| 6 | React Router - rutas estáticas: formularios, carga de imágenes |
| 7 | Rutas dinámicas y protegidas: configuración de Router, navegación con links |
| 8 | Context API: creación de rutas dinámicas, protección de rutas, despliegue inicial |
| 9 | Autenticación de usuarios: login, manejo de estado global con Context API |
| 10 | Creación de productos: conexión a Firebase, validación de formularios |
| 11 | Leer, actualizar y eliminar productos |
| 12 | CRUD completo: integración con estado global, validación y manejo de errores |
| 13 | Estilización con Bootstrap o styled-components |
| 14 | Diseño responsivo y UX: mobile-first, mejores prácticas UI/UX |
| 15 | Despliegue de la aplicación: pruebas en producción, configuración |
| 16 | Revisión final y presentación del proyecto de e-commerce |
