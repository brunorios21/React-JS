Clase 3 — Layout en React

Estructura, composición de componentes y estilos (CSS Modules vs global). Guía rápida y lista de archivos clave.

Ver cuestionario Abrir app (dev)

Objetivos

  • Estructurar un proyecto React con Vite.
  • Implementar un `Layout` reutilizable.
  • Usar CSS Modules para estilos aislados.
  • Construir componentes reutilizables (TarjetaProducto, Gallery).

Cómo ejecutar

Desde la carpeta Clase3:

npm install
npm run dev

Abrir http://localhost:5173/ en Chrome para mejores resultados.

Ejercicios implementados

  1. Layout con Header (Nav) y Footer.
  2. TarjetaProducto con CSS Module.
  3. Gallery con 3 tarjetas de ejemplo.
